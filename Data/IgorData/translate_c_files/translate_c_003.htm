Description Описание This is a merge of ComboBox and CheckListBox Это слияние ComboBox и CheckListBox
You can Multi select Items, checking - unchecking them in the dropdown list. Multi Вы можете выбрать элементы, проверка - отключив их в выпадающем списке. For every selected,unselected  item it's value is inserted,deleted in the combo box line separated by comma from the other choices. Для каждого выбранного, невыделенный пункта вставить его значения, удаленные в окне списка строки, разделенные запятыми из другого выбора.
You can select/unselect all with the right click menu. Вы можете выбрать / отменить все с меню правой кнопкой мыши.
You can take the result in 3 different formats (quoted) ready to use in SQL Вы можете принять результат в 3 различных форматах (цитата) готовы для использования в SQL
